# Changelog

All notable changes to `support` will be documented in this file.

## v1.0.0 - 2024-11-15

### Added
- Initial release
- Partner panel for creating and managing support tickets
- Admin panel for managing and responding to tickets
- Support for ticket status management (Open, In Progress, Waiting for Customer, Waiting for Admin, Resolved, Closed)
- Support for priority levels (Low, Normal, High, Urgent)
- File attachments for tickets and replies
- Internal notes feature for admin collaboration
- Ticket assignment system
- Rich text editor for descriptions and replies
- Comprehensive filtering and tab system
- Laravel 12 compatibility
- FilamentPHP 3 integration
